# Holdem Tournament Builder

Проект веб-приложения для организации домашних офлайн-турниров по техасскому холдему.

Идея проекта: организатор заранее настраивает турнир, а во время игры ведёт
текущее состояние - запускает и ставит на паузу таймер, отслеживает текущий
уровень блайндов, отмечает ребаи, вылеты игроков и итоговые результаты.

Сейчас в проекте реализованы доменная логика, сервисный слой,
PostgreSQL-репозиторий, HTTP API, миграции, тесты и веб-интерфейс на Bootstrap.

Веб-интерфейс MVP сгенерирован с помощью ИИ и используется для демонстрации
работы backend-части. Основной фокус проекта - архитектура и бизнес-логика
backend.
